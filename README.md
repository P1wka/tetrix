# tetrix
- **Tetris game in C# console**
- This project writen in C#, you can build it and run it from your terminal

## Requirements

- Visual Studio (.NET 8.0)
- **NOTE: If you use Windows, there is built files in project. You don't have to compile it again.**


## Installation

1. Open your terminal,

2. Clone the repository:

```bash
git clone https://github.com/P1wka/tetrix.git
cd tetrix
```

3. Open ```tetrix.sln``` file via Visual Studio,

4. Build the project,

4. Follow that steps:
   ### For Linux:
   - In Visual Studio terminal, use this; 
   ```bash
   dotnet publish --runtime linux-x64
   ```
   - Then, go to ```tetrix\tetrix\bin\Release\net8.0\linux-x64```
   - To run commands, open the terminal

   ### For Windows:
   - Build the project in Visual Studio
   - Then, find the ```tetrix.exe``` in ```tetrix\tetrix\bin\Debug\net8.0\```


## Commands
- run, play, start, tetrix --> Start the game
- help, -h, --help --> List of commands
- developer, -d, --developer --> Show info about developer
- version, -v, --version  --> Show the version

## Notes
This game and game engine is under the development.
This is the ```v0.0.1``` of tetrix.
